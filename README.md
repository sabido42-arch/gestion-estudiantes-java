# 📚 Gestión de Estudiantes en Java

Aplicación de consola desarrollada en Java para gestionar estudiantes utilizando Programación Orientada a Objetos, Arrays y ArrayList.

---

## ⚙️ Funcionalidades

### Array
- Añadir estudiante
- Ver estudiantes

### ArrayList
- Añadir estudiante
- Ver estudiantes
- Eliminar estudiante
- Buscar estudiante por nombre

---

## 🛠️ Tecnologías utilizadas

- Java
- Eclipse IDE
- GitHub

---

## 📁 Estructura del proyecto
```
src/
├── Estudiante.java
└── Main.java
```
---

## ▶️ Cómo ejecutar el proyecto

1. Abrir el proyecto en Eclipse.
2. Ejecutar la clase `Main.java`.
3. Usar el menú de consola para interactuar.
